News Nerdery

An international meta organization to foster news nerd collaboration and knowledge sharing

Hi! Welcome to News Nerdery. We're fostering news nerd collaboration and knowledge sharing in these two ways:

  • Synchronous chat using Slack. Email Daniel to signup.
  • Knowledge sharing using a Github docs repo. Create an issue if you have a question you want answered, or submit a pull request if you have knowledge to contribute.

Feedback welcome! Open up a Github issue, or tweet @danielbachhuber.

You can also follow @newsnerdery.
